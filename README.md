# Training portuguese conjugation

With this code you can deploy an local app on iOS14. With this app you can train conjugation in portuguese. 

## Available conjugations
### Indicativo
Presente
Perfeito Simples
Imperfeito
Perfeito Composto
Pretérito Mais-que-Perfeito
Futuro Simples
Futuro Composto

### Subjunctivo
Presente
Perfeito Simples
Imperfeito
Pretérito Mais-Que-Perfeito
Futuro Simples
Futuro Composto

### Condicional
Futuro do Pretérito Simples
Futuro do Pretérito Composto


## How to add verbs
In ContentView.swift you can add verbs to the array ```verbArray```.

First element is the verb, second element is the conjugation rule (ar, er, ir or for irregular verbs ser (f.i.)) and third element is German translation. 

Be aware: Not all different conjugation rules are implementend. Feel free to edit the code according to your own needs.
